# 📊 HR Analytics Project | Excel & Power Query

## 📌 Project Overview

This project is an **HR Analytics solution** built using **Microsoft Excel and Power Query**.
It focuses on cleaning, transforming, and analyzing HR data to generate meaningful insights for decision-making.

The project demonstrates strong skills in **data preparation, transformation, and analysis** without relying on external BI tools.

---

## 🎯 Objectives

* Analyze employee attrition (turnover)
* Understand salary distribution
* Evaluate employee demographics
* Track employee experience (years at company)

---

## 📂 Dataset

* File: `the final analyst.xlsb`
* Includes:

  * Employee ID
  * Age
  * Department
  * Job Role
  * Salary
  * Attrition
  * Gender
  * Years at Company
  * Education Level
